# About the App

FocusFlow is designed to help students maintain focus during long study sessions.

## Key Features

- **Customizable Timers**: Set your own work/break intervals.
- **Analytics**: Track your study hours over time.
- **Minimalist Design**: No distractions, just focus.

## Tech Stack

- **SwiftUI**: For the user interface.
- **CoreData**: For local data persistence.
- **WidgetKit**: Home screen widgets.
