# About NightGuard

**NightGuard** is more than just a sleep tracker; it's a comprehensive habit builder designed to improve your sleep hygiene.

## Key Features

- **Sleep Tracking**: Seamlessly integrates with HealthKit to analyze sleep patterns.
- **Habit Formation**: Set and track pre-sleep rituals to ensure better rest.
- **Data Privacy**: All data is processed locally on the device, ensuring user privacy.

## Technical Details

Built entirely with **SwiftUI**, NightGuard leverages the latest iOS features to provide a smooth, native experience.

- **HealthKit Integration**: For reading and writing sleep data.
- **Local Notification**: Smart reminders for bedtime habits.
- **Swift Charts**: Beautiful visualization of sleep trends.
